## 📝 Sobre o Projeto
Este é um projeto desenvolvido como trabalho final para a disciplina de **Introdução à Programação** do 1° Semestre do curso de **Ciência da Computação**. 

O projeto consiste na implementação do clássico jogo **Lig4** utilizando a linguagem de programação **Java**, sendo executado e jogado diretamente através do terminal da IDE.

## 🕹️ Execução do Jogador (Como Jogar)
* O usuário joga contra o **computador** em um tabuleiro de dimensão **6x7**.
* Ao escolher uma coluna para jogar, a peça vai cair até o ponto mais baixo disponível do tabuleiro.
* O primeiro a fazer uma sequência de **4 peças consecutivas** (seja na horizontal, vertical ou diagonal) ganha o jogo.

## 🛠️ Habilidades Técnicas Aplicadas
Para a construção da lógica e execução do projeto, foram utilizadas diferentes habilidades e conceitos de desenvolvimento, tais como:
* 🔹 **Estruturas Condicionais e de Repetição** (para validações de jogadas, renderização do tabuleiro e verificação de vitória).
* 🔹 **Tratamentos de Erro** (`try-catch` para garantir que o jogo não trave caso o usuário digite algo inválido).
* 🔹 **POO (Programação Orientada a Objetos)** (organização do código em classes e objetos estruturados).
